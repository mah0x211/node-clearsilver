#node-clearsilver

ClearSilver template engine bindings for node.js

## Dependency

* [ClearSilver 0.10.5](http://www.clearsilver.net/)

## Installation

    // install ClearSilver
	export CPPFLAGS="-I/installed/ClearSilver/header/dir $CPPFLAGS"
    npm install ClearSilver
